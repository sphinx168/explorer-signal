# explorer-signal

「探索者1號：訊號接收」是一個用純 HTML、CSS、JavaScript 做成的迷你瀏覽器遊戲。

玩家控制探索者1號左右移動，在 60 秒內接住落下的訊號點，為第一個 GitHub repo 留下一次簡單的發射紀錄。

## Play

直接開啟 `index.html`，或用任一靜態伺服器啟動資料夾。

## Controls

- 方向鍵或 `A` / `D`：左右移動
- 滑鼠或觸控拖曳：控制探索者位置
- `Space` / `Enter`：開始或重新開始

## Files

- `index.html`：頁面結構
- `style.css`：復古太空視覺
- `script.js`：遊戲邏輯
